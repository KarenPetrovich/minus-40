# Tables

## public.app_users

Stores one cloud profile row per Telegram user together with global weight settings.

| Column | Type | Notes |
| --- | --- | --- |
| id | uuid | Primary key generated by Postgres. |
| telegram_user_id | bigint | Unique Telegram user identifier used for account mapping. |
| telegram_username | text | Optional Telegram username snapshot. |
| telegram_first_name | text | Optional Telegram first name snapshot. |
| telegram_last_name | text | Optional Telegram last name snapshot. |
| start_weight | numeric(5,1) | Starting weight for the journey. |
| target_weight | numeric(5,1) | Target weight for the journey. |
| migrated_at | timestamptz | Set when legacy local data is first imported into Supabase. |
| last_seen_at | timestamptz | Updated when the app successfully reaches the cloud layer. |
| created_at | timestamptz | Row creation timestamp. |
| updated_at | timestamptz | Row update timestamp maintained by trigger. |

## public.weight_entries

Stores the user's complete history of weight measurements.

| Column | Type | Notes |
| --- | --- | --- |
| id | text | Primary key that preserves existing local entry IDs. |
| user_id | uuid | Foreign key to app_users.id. |
| measured_at | timestamptz | When the measurement happened. |
| weight | numeric(5,1) | Measured body weight in kilograms. |
| source | text | Either manual or migration. |
| created_at | timestamptz | Row creation timestamp. |
| updated_at | timestamptz | Row update timestamp maintained by trigger. |

## public.comments

Stores one comment per target, either a control milestone or a weight entry.

| Column | Type | Notes |
| --- | --- | --- |
| id | uuid | Primary key generated by Postgres. |
| user_id | uuid | Foreign key to app_users.id. |
| target_type | text | Either milestone or weight_entry. |
| target_key | text | Milestone value or weight entry id. |
| text | text | User comment content. |
| created_at | timestamptz | Row creation timestamp. |
| updated_at | timestamptz | Row update timestamp maintained by trigger. |
